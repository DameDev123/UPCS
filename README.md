# Undergraduate Project Coordination System (UPCS)

A full role-based web application built with PHP & MySQL.

## Features
- Role-based authentication (Student, Supervisor, Coordinator, Admin)
- Project submission and tracking
- Supervisor approval workflow
- Feedback and grading system
- Report generation

## Tech Stack
- PHP
- MySQL
- HTML5, CSS3, JavaScript
